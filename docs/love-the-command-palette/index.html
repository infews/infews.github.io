Don't Sleep on the Command Palette

21 March 2021 — under 1 minute read

While I’m still learning how to use Obsidian, I find the Command Palette amazing. Ctrl+P/Cmd+P brings up a searchable list of commands. I find new actions here all the time. And it’s the fastest way to put an Obsidian URL on your clipboard.
