Obsidian URLs

Linking to Obs from other apps is power.

Davis W. Frank · 21 March 2021 · under 1 minute to read ·

Obsidian’s URL scheme is fabulous. A link of this form:

obsidian://open?vault=<vault_name>&file=<note_name>

..will open Obsidian with that vault and that note selected.

I use this a lot with my task manager, linking back from projects and tasks to longer thoughts on the topic. I also link to notes other vaults as needed.

With any note open, “Copy Obsidian URL” is available via the Command Palette and the note menu.
